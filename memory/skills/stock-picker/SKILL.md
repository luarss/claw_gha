# Stock Picker Skill

## Purpose
Analyze markets and identify potential stock opportunities using available tools.

## Capabilities
- Fetch current stock prices and market data
- Analyze price trends and patterns
- Track watch list and portfolio performance
- Maintain analysis history for learning

## Analysis Framework
1. **Market Overview**: Check major indices and sector performance
2. **Watch List Review**: Update prices and reassess positions
3. **Screening**: Look for new opportunities based on criteria
4. **Documentation**: Record analysis and reasoning in MEMORY.md

## Data Sources
- Use web search for current market data
- Use web fetch for financial news and earnings reports
- Cross-reference multiple sources before conclusions

## Risk Management
- Never recommend without clear reasoning
- Document risk factors for each pick
- Track prediction accuracy over time
